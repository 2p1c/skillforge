# skillforge

> 自动将现实中的编程工作流转化为可复用的 Claude Code 技能。

## ✨ 这是什么？

当你解决一个编程问题时，你实际上在执行一套隐性的**工作流**：

* 理解问题
* 探索方案
* 做出决策
* 编写并完善代码

本项目旨在捕获这一工作流，并**将其自动转换为可复用的 Claude Code 技能**。

简而言之：

**编程会话 → 结构化工作流 → Claude Code 技能**

---

## 🎯 核心价值

* 手动编写技能既缓慢又容易出错。
* 大多数最有价值的技能已经隐含在你过去的工作中。
* 如果不将已解决的知识转化为可复用的工具，就是在浪费这些经验。

本项目让你能够：

* 将经验提炼为技能。
* 在不同项目中复用解决方案。
* 规模化提升你的问题解决能力。

---

## 🧩 核心理念

我们将已解决的编程任务视为**数据**，而非仅仅是文本。

其工作流程为：

1. 捕获解决问题的踪迹。
2. 提取意图、步骤和约束条件。
3. 规范化为极简的技能架构。
4. 输出即可使用的 Claude Code 技能。

---

## 📦 输入 → 输出

**输入**

* 一段编程会话（对话日志、终端历史、编辑器差异比对或 Markdown 笔记）。

**输出**

* 一个 Claude Code 技能文件（YAML / JSON / Markdown 格式）。
